# IAU-project

## run env 
Need to be in folder of iau_prj_env
Linux/Mac: 
```
source ./bin/activate
```
Win: 
```
bin\activate.bat
```

### how to export python packages
```
pip freeze > requirements.txt
```

### how to install requitemnts
```
pip install -r requirements.txt
```

#### create env
```
python3 -m venv iau_prj_env 
```
